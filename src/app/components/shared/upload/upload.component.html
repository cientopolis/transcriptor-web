Tips
  1. ZIP files may contain folders containing images, PDFs, or folders containing pdfs.
  2. Each folder will be treated as a different document, so do not mix pages from different documents in the same folder.
  3. Each PDF will be treated as its own document, so do not split pages from the same document among more than one PDF.
  4. For example a ZIP file with 3 images, 2 PDFs, and 1 folder containing 5 more images would create 4 works: the top level images in one, each PDF in their own work, and a last work containing the 5 images from the folder.
  5. To specify metadata along with images, include a metadata.yml file in each folder specifying fields for the work.
