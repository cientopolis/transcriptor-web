• Paso 1
    Elegí el tipo de dato

    Los tipos de datos te ayudarán a definir mejor lo que querés representar dentro del manuscrito, usa el buscador para encontrar el que necesitas ¡Hay un montón!

  • Paso 2
    Cargá las data properties!

    Ya seleccionaste el tipo de dato, cada uno de estos tiene asociados varias data properties que te van a servir para describir cuales son las características especificas de tu elemento. A partir de ahora podés agregar y cargar las data properties que consideres mas adecuadas.

  • Paso 3
    Cargá las Relaciones

    Llego el momento de generar las asociaciones de tu elemento, estas permitirán generar una descripción mas profunda de su interaccion con el mundo real. Para cargarlas, primero tenes que establecer de que manera se relacionan con tu objeto a través de la opción "Agregar relación", luego solo es cuestión de que referencies a otro elemento ya cargado o cargar uno nuevo indicando sus propiedades.

  • Paso 4
    Revisá y Guardá

    Ahora revisá que los datos sean correctos. Si todo esta bien ya podes continuar con el guardado.
